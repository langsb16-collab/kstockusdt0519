# K-STOCK USDT (USDT-based Korean Stock Platform)

## Setup Guide

1. Clone the repository.
2. Install dependencies: `npm install`
3. Set your `GEMINI_API_KEY` in `.env`.
4. Run development server: `npm run dev`

## Core Features
- **USDT Based Trading**: Trade K-Stocks directly with USDT.
- **No Signup Required**: Uses local persistence for trading experience.
- **Multilingual**: Supports 9 languages.
- **AI Powered**: AI stock analysis and FAQ.
- **Dual Chat System**: Support chat and AI automated FAQ.
- **Premium UI**: Glassmorphism and specialized Day/Night themes.
